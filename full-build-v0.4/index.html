How it works

  1. Load approved copy — upload a PDF/Word file, paste the text, or point to a shared Google Doc.
  2. Enter the page URL and pick a scope (main content, nav, footer, CTAs, links).
  3. Run QA Check — the tool extracts visible text from the page and compares it block-by-block.
  4. Review results — prioritized table of missing copy, mismatches, extra sections, acronym and CTA issues.
  5. Export the report as CSV or copy it to paste into Asana / a ticket.

Note: browsers block cross-origin requests. Auto-fetch uses a public CORS proxy; if it fails (auth-walled or blocked pages), switch Fetch method to Paste page HTML / text.
